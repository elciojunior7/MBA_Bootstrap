# Repositório de FrontEnd

Este repositório contém um projeto de página landscape com o tema "Billiards - Venda de Mesas de Bilhar".

Layout foi editado usando a ferramenta Figma (https://www.figma.com/file/Rvxt7vsCFIj4ShAmtTexxt/Billiards?node-id=0%3A1).
Então, a página foi estruturada usando HTML, CSS e Bootstrap.
